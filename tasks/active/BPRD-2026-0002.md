## Description

Automate the handoff lifecycle as defined in the V2 protocol:
1. Created → Active → Acknowledged → Complete
2. 4-hour acknowledgment window enforcement
3. Per-agent caps on active handoffs
4. Automatic archival of completed handoffs

verification_criteria:
1. Handoff state transitions work correctly
2. Acknowledgment window alerts are generated
3. Agent handoff caps are enforced
4. Completed handoffs are automatically archived
