## Description

Define skill-web nodes in `_shared/skill-web/skills/` for:
1. Core skills (api-development, system-architecture, etc.)
2. Prerequisites and related skills mapping
3. Agent capability profiles
4. Skill gap escalation rules

verification_criteria:
1. All core skills have node definitions
2. Prerequisites are correctly mapped
3. Agent profiles link to skill nodes
4. Skill gap escalation works

## Notes

This enables proper task-to-agent assignment based on skills.
