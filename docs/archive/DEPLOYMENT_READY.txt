HOW IT WORKS
================================================================================

WEEKLY CYCLE (Per Agent - 75 minutes/week)
  Sunday EOD:    Agent reviews week (30 min)
  Monday AM:     Agent updates 4 memory files (45 min)
  Tuesday AM:    Agent posts summary in chat (15 min, optional)
  Ongoing:       Agent references memories for decisions

MONTHLY CYCLE (Month End - 30 minutes)
  All agents propose 1-3 items for shared memory
  You review & merge proposals
  Update AGENT_PATTERNS.md, MEMORY.md as needed

PAYOFF TIMELINE
  Week 1-2:  Setup cost (agents learning system)
  Week 3-4:  First visible benefits
  Month 2+:  Compound returns (10+ hours/week saved)
