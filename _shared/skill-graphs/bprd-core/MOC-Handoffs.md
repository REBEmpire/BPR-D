## Handoff System Overview

BPR&D uses a **two-tier handoff system**:

| Tier | Location | Purpose | Format |
|------|----------|---------|--------|
| Operational | `_agents/[agent]/handoff.md` | Day-to-day tasks | Markdown table |
| Escalation | `_agents/_handoffs/*.md` | Multi-session, Russell-involved | Full template |
