## Cooldown Rules

- No agent runs back-to-back slots (build in variety)
- Grok ALWAYS runs the slot immediately before Daily Briefing (or 2 slots before) for pre-briefing prep
- Abacus: check `ABACUS_ACTIVE` env var; if False, skip all Abacus slots and redistribute

## APScheduler Config Pattern

```python
AGENT_SLOTS = [
    ("00:00", "grok"),
    ("02:00", "claude"),
    ("04:00", "gemini"),
    # ... etc
]

for time_str, agent in AGENT_SLOTS:
    hour, minute = map(int, time_str.split(":"))
    scheduler.add_job(
        run_work_session,
        'cron',
        hour=hour,
        minute=minute,
        timezone='US/Pacific',
        kwargs={"agent": agent},
        id=f"work_session_{agent}_{hour:02d}{minute:02d}",
        replace_existing=True,
    )
```
