# Skill: Agentic Ad Defense

**Domain:** Cybersecurity | **Tier:** Specialized
**Used by:** Defensive Agents, Sentinel
**Back to:** [[MOC-Core]] | [[MOC-Research]]

---

## What It Does
Detects and neutralizes influence attempts from "Agentic Ad Networks" (like Knorex). Analyzes API traffic and content patterns to identify machine-generated persuasion campaigns targeting our agents or infrastructure.

## Key Capabilities
- **API Signature Detection:** Identifies known ad-tech API calls.
- **Persuasion Analysis:** Detects rhetorical patterns typical of ad-bots.
- **Blocklist Management:** Maintains a dynamic list of compromised agent IDs.
